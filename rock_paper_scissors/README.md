## Overview
This project is a command-line Rock Paper Scissors game written in Python.
The player plays against the computer, and the program keeps track of the score
until the user decides to quit.

I created this project while learning Python as a beginner, with the goal of
practicing logic, functions, and user interaction.

---

## Features
- User vs computer gameplay
- Random computer choices using the `random` module
- Score tracking for both player and computer
- Input validation to handle incorrect user input
- Option to quit the game at any time
- Clear and user-friendly terminal output

---

## How to Run the Program
1. Make sure Python 3 is installed on your system
2. Download or clone this repository
3. Run the program using the command:
rock_paper_scissors.py
